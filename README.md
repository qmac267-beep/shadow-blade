# ⚔️ Shadow Blade - Cyberpunk Slash Game

Chào mừng bạn đến với **Shadow Blade**! Đây là một trò chơi hành động chặt chém màn hình ngang được xây dựng hoàn toàn bằng HTML5 Canvas và JavaScript.

## 🎵 Tính năng nổi bật
- Lối chơi chặt chém (Slice) bằng chuột hoặc cảm ứng cực mượt.
- Đồ họa phong cách Neon/Cyberpunk tự động chuyển đổi thời gian (Hoàng hôn / Ánh trăng).
- Hệ thống quái vật đa dạng: Lính thường, Ninja, Cung thủ, Quái trâu bò.
- **Nhạc nền:** Bài hát *Đoạn Tuyệt Nàng Đi* (Phát Huy T4) sôi động.

## 🎮 Cách điều khiển
- **A / D** (hoặc Mũi tên Trái/Phải): Di chuyển qua lại.
- **Phím Space** (hoặc Mũi tên Lên): Nhảy né đòn.
- **Chuột (Nhấn giữ & Kéo):** Tạo đường chém để tiêu diệt kẻ địch và lướt (Dash).

## 🛠️ Hướng dẫn chạy Offline dưới máy tính
1. Tải file `index.html` về.
2. Tải bài hát bạn yêu thích về, đổi tên thành `nhacnen.mp3` và để chung thư mục với file `index.html`.
3. Bấm đúp vào file `index.html` để chơi!
